### 1. MÓDULO DE TECLADO (KEYBED & ACTION)
*(La parte más mecánica y propensa a desgaste)*

#### 1.1 Teclas
*   **Tecla Blanca** - Plástico (varias longitudes/despuntes según octava)
*   **Tecla Negra** - Plástico (varias formas: cuadrada, estilo piano, estilo Hammond)
*   **Tapas de Tecla** - Intercambiables (Para teclas dañadas)

#### 1.2 Mecanismo de Contacto
*   **Matriz de Goma (Rubber Dome)** - Contactos de carbón (Switch de membrana)
*   **Buscador de Contacto (Keypad Contact Strip)** - PCB metálica con domos
*   **Pasta de Restauración** - Para contactos de goma oxidados (Graphite)
*   **Contactos de Oro (Gold Contacts)** - Para mecanismos pesados
*   **Placa PCB de Escaneo** - Debajo del teclado (Matrix board)

#### 1.3 Mecanismo de Acción (Pianos Ponderados/Sintetizadores)
*   **Martillo (Hammer)** - Acción ponderada (Madera o plástico)
*   **Palanca de Gatillo (Whippen)** - Mecanismo interno
*   **Resortes de Retorno** - Para teclas
*   **Feltro (Fieltro)** - Para amortiguación (varios grosores)
*   **Contrapesos (Balance Weights)** - Plomo insertado en teclas frontales
*   **Stop Felt** - Fieltro de parada
*   **Bumpers de Silicona** - Tope inferior de tecla

### 2. INTERFAZ DE USUARIO (CONTROLES)
*(Todo lo que el músico toca para editar sonido)*

#### 2.1 Perillas y Mandos (Knobs)
*   **Perilla Push-On** - Eje estriado (Splined), estilo Moog/Roland
*   **Perilla de Tornillo** - Eje liso (D-Shaft), con set screw
*   **Perilla Tipo "Skirted" (Falda)** - Cubre el eje
*   **Perilla Golpeada (Knurled)** - Textura metálica
*   **Perilla Slider** - Tapa de plástico para faders
*   **Tapa de Encoders** - Often smaller or specific shape
*   **Tampón / Botón de Goma (Rubber Button)** - Botones flexibles membrana
*   **Indicador de Nivel (Pointer)** - Pequeña flecha de plástico sobre el pote

#### 2.2 Potenciómetros y Faders
*   **Pote Lineal (Slide Fader)** - 10k, 50k, 100k (Canales, Volumen maestro)
*   **Pote Rotatorio** - Lineal (A), Logarítmico (C/B), Audio Taper
*   **Encoder Rotativo Incremental** - Sin fin (Para control de parámetros digital)
*   **Encoder con Push** - Encoder que también se presiona
*   **Pote Dual / Estéreo** - Para control panorámico
*   **Motorized Fader** - Fader con motor (Automatización)

#### 2.3 Interruptores y Selectores
*   **Botón de Panel (Panel Mount Switch)** - Momentáneo o Latch
*   **Switch de Palanca (Toggle Switch)** - SPDT, DPDT
*   **Switch de Palanca Miniatura** - Para funciones internas
*   **DIP Switch** - Configuración de sistema (a veces externo)
*   **Switch de Pala (Rocker Switch)** - Power o funciones

### 3. MÓDULO DE EXPRESIÓN
*(Control de pitch y modulación extra)*

#### 3.1 Ruedas y Joysticks
*   **Ensamblaje de Rueda (Pitch Bend/Mod Wheel)** - Plástico, eje, óptico o potenciómetro integrado
*   **Tapa de Rueda** - Parte exterior que se desgasta
*   **Resorte de Retorno** - Para la rueda de Pitch Bend
*   **Joystick (Stick Controller)** - Asignable (X/Y)
*   **Potenciómetro de Joystick** - Cermet (gimbal)

#### 3.2 Pedales y Conexión
*   **Jack de Pedal (Sustain/Expression)** - 1/4" TS o TRS (Chasis o PCB)
*   **Inversor de Polarity Switch** - Para pedal de sustain (en el cable o panel)
*   **Pedal de Expresión** - Externo (Pieza de repuesto)
*   **Pedal de Sustain** - Externo (Pieza de repuesto)

### 4. VISUALIZACIÓN Y PANEL
*(Pantallas e indicadores)*

#### 4.1 Pantallas (Displays)

*   **TFT LCD / OLED** - Pantallas a color modernas
*   **Pantalla VFD** - Tubo de vacío fluorescente (Clásicos Nord/Yamaha)

#### 4.2 Indicadores (LEDs)

*   **LED Transparente** - Para botones con ventana
*   **LED SMD** - Soldered directly to PCB

*   **Electroluminiscente (EL Backlight)** - Láminas de luz

*(Carrocería y soporte)*

#### 5.1 Carrocería Externa

#### 5.2 Ferretería de Montaje

*   **Soporte (Standoff)** - Espaciador Hembra-Hembra o Macho-Hembra

#### 5.3 Soporte Interno

    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    
    FOREIGN KEY (brand_id) REFERENCES tool_brands(id),
    FOREIGN KEY (category_id) REFERENCES tool_categories(id),
    FOREIGN KEY (location_id) REFERENCES storage_locations(id)
);

-- HISTORIAL DE MANTENIMIENTO DE HERRAMIENTAS
CREATE TABLE tool_maintenance_logs (

    performed_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    next_action_date DATE,
    FOREIGN KEY (tool_id) REFERENCES tools(id)
);

CREATE INDEX idx_tools_location ON tools(location_id);
